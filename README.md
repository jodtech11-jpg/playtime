# Playtime

Repository connected to [GitHub](https://github.com/jodtech11-jpg/playtime).

## Pushing your code

When Git is installed, from this folder run:

```powershell
git add .
git commit -m "Initial commit"
git push -u origin main
```

Or use **Source Control** in Cursor (Ctrl+Shift+G): stage changes, commit, then push.
